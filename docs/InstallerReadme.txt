To start this installer :

- double click on the file "photomailwizard-installer-1.0.1.jar" in order to start installation
- or do "Open With" and select "Java Platform ..."
- or run the following command : javaw -jar photomailwizard-installer-1.0.1.jar

If you do not have Java 5 already installed, please
visit : http://java.com
